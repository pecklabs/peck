# Quest mascot images

Drop PNG files here with these exact names. If a file is missing, the app falls
back to an emoji, so you can add them incrementally.

Spec per image:
- **PNG, transparent background**, full color (these are content illustrations, not menu-bar template icons).
- Square. Provide ~**88×88 px** (it's displayed ~24pt and scaled down) for crispness.

Progression: 🥚 egg → 🐣 chick → 🐔 chicken(닭) → 🍗 fried chicken(치킨, done).
Changes requested forks off to 🍳 fried egg.

Base stages (no conflict):
- `egg.png`          — 🥚 Laid (0 approvals)
- `chick.png`        — 🐣 Hatching (some approvals)
- `chicken.png`      — 🐔 Grown / one approval away
- `friedchicken.png` — 🍗 fully approved — 치킨!
- `friedegg.png`     — 🍳 changes requested

Conflict variants (PR has a merge conflict) — same names + `-conflict`:
- `egg-conflict.png`          — cracked/broken egg
- `chick-conflict.png`        — crying chick
- `chicken-conflict.png`      — ruffled chicken
- `friedchicken-conflict.png` — messed-up 치킨
- `friedegg-conflict.png`     — broken fried egg
